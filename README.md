# 💫 About Me:
Soy **CEO de Scalanze** y un apasionado de la ingeniería digital. Mi enfoque combina el desarrollo de software de alto nivel con la optimización estratégica (SEO/GEO) y la automatización de procesos. Me especializo en convertir problemas complejos en soluciones escalables, liderando equipos técnicos y manteniendo un estándar de ejecución impecable.

Actualmente, estoy profundizando en la **Ingeniería de Ciberseguridad** y el desarrollo móvil moderno.

- 🚀 **Proyectos actuales:** Desarrollo de aplicaciones móviles nativas y ecosistemas digitales automatizados.
- 🛠️ **Enfoque técnico:** Arquitecturas limpias, optimización de rendimiento (PageSpeed) y automatización con n8n/Power Automate.
- 🎓 **Formación:** Estudiante de Ingeniería en Ciberseguridad en Saint Leo University.
